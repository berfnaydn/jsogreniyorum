Tip Dönüşümleri

JavaScript tip dönüşümlerini otomatik olarak kendisi yapabildiği gibi number ile number türüne, String ile String türüne, Boolean ile Boolean türüne dönüşüm yapabilir.
Sayı veri türünü metin veri türüne dönüştürme : Sayıları metin veri türüne dönüştürebilmek için String fonksiyonu kullanılır.
var sayi = 1234; alert(String(sayi)); alert(String(123)); alert(String(100 + 23));
: Sayı ver türünün toString() fonksiyonu sayıları metin veri türüne döönüştürebilir.
var sayi = 14;
alert(sayi.toString());
alert((1213).toString());
alert((1300 + 243).toString());
